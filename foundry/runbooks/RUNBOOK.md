# JasonOS Foundry Runbook

## User instruction model
- WAIT means no action.
- CHATGPT REQUIRED means type 1 in ChatGPT.
- JASON ACTION REQUIRED means run the shown script or follow the shown action.

## Enabled low-risk lanes
- Docs and runbooks.
- Dashboard and status wording.
- Backlog and memory files.
- Architecture review outputs.
- Technical debt reports.

## Locked lanes
- ScarFLIX media workflow changes.
- Catalogue expansion.
- Plex library changes.
- Destructive pruning.
- Secrets access.
