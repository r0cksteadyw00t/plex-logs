# ScarFLIX Foundry Status View

Purpose: compact status page for ScarFLIX Foundry.

Main fields:

- mission status
- user required yes or no
- current objective
- last action
- next action
- visible count
- runner count
- last update

Telemetry files:

- latest/scarflix_v2/status_compact.json
- latest/scarflix_v2/CLOSED_LOOP_SUPERVISOR_STATUS.json
- latest/scarflix_v2/command_result.json

Design rule: main status must fit on one screen. Detailed logs should be linked separately.
